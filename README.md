# coinbase-pro-stats
A simple and fun code to list products and stats from the Coinbase API with React Native

## First, install Node and Expo
- Node 12.4.0 -> https://nodejs.org/en/
- Expo: `npm install -g expo-cli`

## How to run
Add dependencies with `yarn`
then `yarn start`

## How to test
`yarn test`
